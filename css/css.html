文本垂直居中 #

  • 单行文本：行高等于容器高度
  • 由于table单元格默认垂直居中，据此实现容器内多行文本垂直居中
html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Document</title>
  <style>
    div {
      display: table;
      width: 100px;
      height: 100px;
      font-size: 12px;
      border: 1px solid #000;
    }

    span {
      display: table-cell;
      vertical-align: middle;
    }
  </style>
</head>
<body>
  <div>
    <span>你好，蓝轨迹！你好，蓝轨迹！你好，蓝轨迹！</span>
  </div>
</body>
</html>
